Portfolio

Tank Turn Up

Link to Repo

Timeframe: 3 Months

Software Used: Unreal Engine 4 | AutoDesk Maya | Piskel

Role: Game Developer

Skills: C# | 3D Modelling | Animation | Pixel Art

Tank Turn Up is a 3D platformer that was made in Unreal Engine 4 using Blueprints and Maya 3D modelling software. In this game you play as a tank and collect points whilst dodging different obstacles and accumulating a high score. This project was made for my FYP in college and it tested my skill with 3D modelling, the Unreal Engine Blueprint API and my Game Design knowledge. Overall I love making the game and further refinement would make the game even better.

Cat Rush

Link to Repo

Timeframe: 5 days

Software Used: VS Code | Raylib | Piskel

Role: Game Programmer

Skills: C++ | Pixel Art

Cat Rush is a 2D game that was made using a graphical 2D coding library called Raylib. This game was made with C++ and VS Code. This was a mini project that I used to test my knowledge on C++ and creating functionality for a game. I have done multiple mini projects however this is one of the most recent ones I have done.

Within the game you play as a cat that has to rush to collect the different food items within the level. Once all the food items are collected, the level resets and you have to collect less food items than before however the time has gone down so you need to race against the time to make sure that cat does not perish!.

Marble Madness

Link to Repo

Timeframe: 6 Weeks

Software Used: Unity Game Engine | Visual Studios

Role: Game Developer

Skills: C# | Pixel Art

Marble Madness is a 3D Marble game made in Unity game engine. The main premise of the game is to collect points, get over the obstacles and get to the goal located at the end of the level. Each level gets harder than the last and you need to use what you have learnt from previous levels to beat the last one (Which is very hard in my opinion), however, it is possible to finish the game in a good time. I created this as one of my projects for an assignment in University. We were given a framework and we had to adapt to suit how we visualised it.

Back-Track (Game Jam)

Link to Repo

Timeframe: 2 Days

Software Used: Unity Game Engine | Piskel, Visual Studio

Role: 2D Pixel Artist

Skills: Pixel Art

Back-Track is a 2D Pixel Art puzzle game where the player has to traverse different levels and complete different challenges to progress. You play as a small grandfather clock character with the ability to clone its self into a stopwatch character with parallel movement. You must find ways of using the clone and the original to beat these challenges. As well as this, you are able to "Back Track" by using an ability that allows you to swap yourself with a clone you create.

This game was created as part of the Global Game Jam event that me and two other of my friends had participated in. I had decided to take on the art as I was one of the best equipped for the job. The jam was a very fun experience that I would recommend everyone try at least one as it allows you to have fun and work within a team and do roles you are suited to do and want to do.
